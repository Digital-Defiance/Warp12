# Class I* model artifacts (lazy-loaded)

Generated by the training pipeline — not committed to git.

```bash
yarn class1-star:setup      # once: Python venv
CLASS1_STAR_GAMES=500 yarn class1-star:collect   # penalty default
yarn class1-star:train
yarn class1-star:bench
```

Produces:

- `class1-star-v1.onnx` — ONNX model (input `features` [N, 303], output `residual` [N, 1])
- `class1-star-v1.json` — TS fallback weights (same schema as `Class1StarModelWeights`)

See `tools/nn/README.md` for details. Use `CLASS1_STAR_OBJECTIVE=go-out` for the alternate objective.
